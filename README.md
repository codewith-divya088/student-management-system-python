# 🎓 Student Management System (Python + SQLite)

## 📌 Overview
This project is a simple Student Management System built using Python and SQLite.

It allows users to store, view, and delete student records using a menu-driven interface.

---

## 💡 Features
- Add student details  
- View all records  
- Delete specific student  
- Database integration using SQLite  

---

## 🧠 Concepts Used
- Python (Functions, Loops)  
- SQLite Database  
- SQL Queries (INSERT, SELECT, DELETE)  

---

## ▶️ How to Run
1. Run main.py  
2. Choose options from menu  
3. Manage student data  

---

## 🎯 Purpose
This project was built to understand how real applications manage and store data using databases.
